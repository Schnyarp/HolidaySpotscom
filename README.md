This repository contains the code for the [HolidaySpots.com](https://holidayspots.com) domain-for-sale landing page.

## Purpose

This landing page informs visitors that HolidaySpots.com is available for acquisition and provides a convenient way for interested buyers to submit inquiries.

## Features

- Modern, mobile-friendly single-page design
- Prominently displays that HolidaySpots.com is for sale
- Inquiry form (integrated with Formspree or your chosen provider) for collecting offers and questions
- Automatically detects and displays the current domain name
- Fast, secure, and easy to deploy (static HTML/CSS/JavaScript)
- Customizable content and styling for your needs

## Usage

1. **Clone or download this repository.**
2. **Customize the content** in `index.html` as needed (domain name, form endpoint, features, etc.).
3. **Deploy to your preferred static hosting provider** (Netlify, Vercel, GitHub Pages, cPanel, etc.).
4. **Update DNS settings** to point HolidaySpots.com to your hosting provider.

## Customization

- Change the domain name in the header and throughout the page as needed
- Update the Formspree endpoint or other contact method to receive form submissions directly
- Adjust styles, features, or text to better suit your branding or preferences

## DNS Setup

For best performance, use a CNAME (for subdomains) or ALIAS/ANAME or A record (for root domains) to point HolidaySpots.com to your host.  
Avoid HTTP redirects, as these can interfere with domain detection and tracking.
